About this demo This course project analyzes EMR audit logs against user access-control rules to flag potential policy violations (explicitly denied actions) and warnings (no matching rule). This custom demo uses preset datasets (A–D) so results are deterministic, while keeping implementation details private per NCSU Academic Integrity guidelines.
